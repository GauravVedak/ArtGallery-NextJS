
# ArtGallery Next

An interactive art gallery web application built with Next.js. Explore, search, and save your favorite artworks. Includes authentication, user history, and personalized favorites.

## Features

- Browse and search artworks
- View detailed artwork information
- User authentication (login/register)
- Save favorites and view history
- Responsive design

## Installation

1. Clone the repository:
	```bash
	git clone <repo-url>
	cd my-app-master
	```
2. Install dependencies:
	```bash
	npm install
	```
3. Start the development server:
	```bash
	npm run dev
	```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

- Register or log in to access personalized features.
- Use the navigation bar to explore, search, view favorites, and history.
- Click on an artwork card for detailed information.

## Folder Structure

- `components/` – UI components (ArtworkCard, Layout, MainNav, etc.)
- `lib/` – Utility functions (authentication, user data)
- `pages/` – Next.js pages (index, login, register, artwork, etc.)
- `public/data/` – Static data (validObjectIDList.json)
- `styles/` – CSS modules and global styles

## Technologies Used

- Next.js
- React
- CSS Modules
